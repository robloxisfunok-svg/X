Interactive Trust Wallet-style website.

Open index.html in a browser.

Features:
- Tap any crypto logo to edit that coin.
- Enter either a crypto amount or a USD value.
- USD values calculate from current CoinGecko prices when internet is available.
- Tokens automatically reorder from highest USD value to lowest.
- Multiple edited coins are saved in the browser.
- Prices refresh every 60 seconds, with screenshot-matching fallback prices when offline.
